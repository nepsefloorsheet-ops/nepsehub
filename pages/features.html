What's Been Built

Dashboard Features

  • NEPSE Index Display Real-time index value with change indicators and percentage
  • Interactive Chart Multiple timeframes (1D, 1W, 1M, 3M, 6M, 1Y) with Chart.js
  • Market Summary Stats Advances, Declines, Unchanged, Positive/Negative Circuits
  • Market Sentiment Gauge Visual indicator showing Fear/Neutral/Greed levels
  • Indices & Sub-Indices All NEPSE sector indices with real-time updates
  • Top Gainers & Losers Dynamic tables showing best and worst performers
  • Top Volume/Turnover/Transactions Tabbed interface for different market metrics
  • Stock Ticker Scrolling ticker with real-time price updates
  • NEPSE Announcements News feed with modal for detailed announcement viewing

Live Market Features

  • Real-time Data Table All stocks with LTP, LTV, Change, Open, High, Low, Volume, Turnover
  • Symbol Search Quick search functionality to find specific stocks
  • Sector Filtering Filter by Commercial Banks, Hydro Power, Insurance, etc.
  • Quick Filters Click on Advances/Declines/Neutral to filter instantly
  • Sticky Header Table header stays visible while scrolling
  • Auto Refresh Automatic data updates with last updated timestamp
  • Scroll Indicators Visual cues for horizontal scrolling on mobile

Sector Rotation Features

  • Price Strength Mode Analyze sectors based on average percentage change
  • Money Flow Mode Track market share and turnover distribution across sectors
  • Sector Detail Panels Deep-dive into specific sectors with live metrics and top performers
  • A/D Ratio Visualization Visual bar chart showing advances, declines, and unchanged stocks per sector
  • Top Sector Performers Instant view of best and worst performing stocks within each sector
  • Cross-Page Navigation Quick filters to jump from sector view to Heatmap or Live Market
  • Auto-Refresh Timer Built-in countdown for real-time market pulse updates

Heatmap Features

  • Treemap Visualization D3.js powered treemap with dynamic sizing based on market activity
  • Turnover View Size boxes based on trading turnover value
  • Volume View Size boxes based on trading volume
  • Color-coded Performance Green for gains, red for losses, with intensity based on magnitude
  • Sector Filter Focus on specific sectors with dropdown filter
  • Hover Tooltips Detailed info on hover including LTP, change, and turnover data
  • Legend Guide Color legend for easy interpretation

UI/UX Features

  • Dark/Light Theme Theme switcher with smooth transitions and persistence
  • Shimmer Loading States Premium loading animations while data loads
  • Pulsating Live Indicator Animated dot showing live data connection
  • Responsive Design Works seamlessly on desktop, tablet, and mobile
  • Market Status Display Shows if market is Open/Closed with visual indicators
  • Real-time Clock Current date, day, and time display in header
  • Roboto Mono Typography Clean, professional trading-style font
  • Premium Glassmorphism Modern glass-like UI elements with blur effects

Technical Features

  • API Integration Live data from NEPSE through proxy API service
  • Error Handling Graceful error handling with user-friendly messages
  • Cleanup Manager Proper resource cleanup for intervals and event listeners
  • CSP Compliance No inline scripts, all event handlers via addEventListener
  • Cache Busting Version query params for updated script loading
  • Modular Architecture Organized JS/CSS files by feature and component
